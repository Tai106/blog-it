# blog-it

## Purpose
* Users are able to sign-up and create a profile, where they can make a blog post, and interact with other users by commenting and liking there post. 

## Blog Features
* Sign-up Page 
* Sign-In / Sign-Out
* Make a post 
* Comment on a post 
* Delete a post
* Like Button

## Tech Used
* Python
* Flask
* HTML
* Jinja
* BootStrap
* SQLAlchemy
